# Practica-LINQ-CI

He desarrollado esta aplicación de consola en C# como parte de la Tarea Práctica 4 de la unidad 4 del modulo 2. El objetivo es poner en práctica el uso de LINQ (Language Integrated Query) sobre una colección de objetos de la clase `Producto`.

## ¿Qué hace la aplicación?

Creé una lista de 50 productos con datos variados y luego resolví 40 ejercicios propuestos, aplicando consultas LINQ. Cada ejercicio cubre operaciones comunes como filtros, ordenamientos, agrupaciones, cálculos de agregación, verificación de condiciones y más. Los resultados se muestran directamente en la consola de forma organizada.

## Cómo ejecutar el programa

1. Clona este repositorio o descarga el código fuente.
2. Abre la solución en Visual Studio 2022 o superior.
3. Compila y ejecuta el proyecto (también puedes usar `dotnet run` desde la terminal).
4. Verás en la consola cada uno de los 40 ejercicios con su respectivo resultado.

## Estructura del código

- **Producto.cs**: contiene la clase con las propiedades `Id`, `Nombre`, `Descripcion`, `Precio`, `Stock`, `Categoria` y `FechaCreacion`.
- **Program.cs**: incluye la generación de la colección de productos y los métodos que resuelven cada ejercicio usando LINQ.
